## git reset
수정된 버전을 삭제하면서 수정전 버전으로 돌아가고 싶을 때 사용한다.
## git revert
수정된 버전을 유지하면서 수정전 버전으로 돌아가고 싶을 때 사용한다.

> 만약  a버전으로 돌아가고 싶을 때는?
1. 현재버전
2. c버전
3. b버전
4. a버전
> c버전 revert → b버전 revert → a버전 revert 식으로 revert해야한다.
> 이렇게 순차적인 과정이 없다면 충돌이 발생한다.
